
Cal State Casino

Cal State Casino is a Python-based casino simulator that includes games like Roulette, Dice Roll, and Blackjack, built using Pygame. It offers a fun and interactive way to experience casino games without risking real money!

## Installation
1. Download and extract the project files.

## Dependencies
If you're running the source code directly:
- Python 3.8 or later
- Pygame library (install using `pip install pygame`)

## How to Play
1. Launch the application.
2. Select a game from the main menu.
3. Follow on-screen instructions for each game.
